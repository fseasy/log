今天又学习了 release-please 这个工具—一套下来，自动发布 release、tag，自动 bump version, 太牛了！
流程是：
1. git commit 的时候，commit 里带上关键的 action: fix, feat, 或者 feat!
2. merge 后，会自动创建一个 PR 用来 release，更改有 CHANGELOG.md, pyproject.toml 里的 version
3. 合并 PR 后，release 会自动创建，tag 也会创建，这时陪的 tag->pypi 的部署也会被触发，太牛了
